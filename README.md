## findProduct - java

자료구조에 중점을 두어 가장 적절한 sort를 사용하는 것이 목적이다.

+ java + DB
+ 개발 기간 : 2016.09 ~ 2016.11
+ 본인의 역할 : 팀원 - 코딩(GUI, EventData), 품목 비교 및 정렬 설계, 검색어 자동완성, 보고서 작성
+ 기능
  + 로그인/회원가입
  + 위치 탐색
  + 문자열 자동 완성 / 정렬
  + 행사 상품
  + 인기 검색 / 즐겨찾기
